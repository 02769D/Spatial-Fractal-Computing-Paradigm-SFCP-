# SFCP：空间分形计算范式

> **基于空间分形自相似性原理：从低维到高维，同一套规则全维度成立。**
>
> 在 CPU 上实现 10⁵⁰⁰⁰ 维稳定收敛。从小学数学到高维科学计算，同一套流程。

## ✨ 核心主张

| 别人的范式 | SFCP 范式 |
|------------|-----------|
| 低维有效，高维失效 | **低维有效（GSM8K 98.86%）** |
| 高维需要换方法 | **高维也有效（10⁵⁰⁰⁰ 维收敛）** |
| 维度越高越不稳定 | **维度越高越稳定（收敛规律不变）** |
| 换场景要重写 | **同一套六步，换数据就行** |

**SFCP 不是“解决高维问题的技巧”，而是一个从低维到高维都成立的数学规律。**

## 🚀 核心验证

| 场景 | 数据规格 | 核心结论 |
| :--- | :--- | :--- |
| **小学数学推理** | GSM8K 1319题 | 98.86%，计算层正确率 100% |
| **黎曼零点收敛** | 10⁵⁰⁰⁰ 维 + 120 级噪声 | 均值稳定收敛于 0.5，噪声无法打破底层规律 |
| **金融波动率建模** | 10⁵⁰⁰⁰ 维（模拟高频数据） | 波动率向行业基准值 0.20 收敛 |
| **文本高维规律** | 10⁵⁰⁰⁰ 维（内置中文语料） | 字符熵向临界熵收敛，语法连贯度趋近于 1 |
| **工业级文档一致性** | **142 万 Token**（真实标注数据） | 与人工标注相关性达 0.95 |

## 🧠 核心机制：SFCP 六步

**定源 → 分裂 → 计算 → 剔除 → 保存 → 映射**

分形结构的无限复杂性，来自极简边界约束下的无限迭代。SFCP 六步的本质是：**不信任任何单一数据源，用规则和确定性计算兜底。**

| 步骤 | 说明 |
|------|------|
| **定源** | 定义边界约束，划定合法空间 |
| **分裂** | 将复杂问题拆解为可独立计算的单元 |
| **计算** | 模型只做翻译，计算完全交给符号引擎 |
| **剔除** | 边界强制执行，含变量、语法错误直接斩杀 |
| **保存** | 版本化快照，每次运行都有完整存档 |
| **映射** | 将计算结果映射回原始问题结构 |

git clone https://github.com/02769D/SFCP.git
cd SFCP

# 2. 安装依赖（仅需基础科学计算包）
pip install numpy scipy matplotlib pandas scikit-learn tqdm

# 3. 直接运行任意验证脚本
python 升级版1黎曼.py      # 10⁵⁰⁰⁰ 维黎曼零点收敛
python 扩展金融测试1.py    # 10⁵⁰⁰⁰ 维金融波动率
python 文本生成规律.py      # 10⁵⁰⁰⁰ 维文本规律
python 空间序222.py        # 142 万 Token 文档一致性
所有脚本内置数据生成逻辑，无需额外配置，运行后自动生成可视化图表与结果文件。
